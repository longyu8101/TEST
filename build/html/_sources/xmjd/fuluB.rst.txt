附录B 检测与评估报告编写要求
================================

.. raw:: html

 <p><a href="#idB.0.1"> B.0.1</a> <span id="idB.0.1">检测报告的编写应符合下列规定。</span></p>
 <p><a href="#idB.0.1.1"> B.0.1.1</a> <span id="idB.0.1.1"> 检测报告应主要包括下列内容：</span></p>
 
 <ol>
 <li>检测报告的标题；</li>
 <li>检测单位名称、地址、联系人和联系方式；</li>
 <li>检测报告唯一性标识(编号)；</li>
 <li>检测单位的资格能力证明；</li>
 <li>项目负责人、检测人员、报告编写人、审核人、批准人；</li>
 <li>检测项目的概况,主要包括工程名称、地点、规模、结构型式、修建年代、检测目的和检测日期等；</li>
 <li>检测依据,主要包括检测项目合同、相关的现行国家或行业标准、设计图纸、竣工资料和其他有关的技术文件名称；</li>
 <li>检测内容和方法；</li>
 <li>检测所用的仪器设备,主要包括仪器设备名称、型号、唯一性(编号)标识、主要性能指标及有效状态；</li>
 <li>检测结果；</li>
 <li>检测结论。</li>
 </ol>
 
 <p><a href="#idB.0.1.2"> B.0.1.2</a> <span id="idB.0.1.2">检测报告应由检测人员编写；提交的报告应经审核人的审查核实和批准人的审查批准,并应有编写人、审核人和批准人的签字确认。</span></p>
 <p><a href="#idB.0.1.3"> B.0.1.3</a> <span id="idB.0.1.3">检测报告的编写人、审核人和批准人应是具有相应资格能力的试验检测技术人员。</span></p>
 <p><a href="#idB.0.2"> B.0.2</a> <span id="idB.0.2">评估报告的编写应符合下列规定。</span></p>
 <p><a href="#idB.0.2.1"> B.0.2.1</a> <span id="idB.0.2.1">评估报告应主要包括下列内容：</span></p>

 <ol>
 <li>评估报告的标题；</li>
 <li>评估单位名称、地址、联系人和联系方式；</li>
 <li>评估报告唯一性标识(编号)；</li>
 <li>评估单位的资格能力证明；</li>
 <li>项目组成员、报告编写人、审核人和批准人；</li>
 <li>评估项目的现况,主要包括工程名称、地点、设计与施工概况、自然条件、总平面布置、结构型式、运营情况等;</li>
 <li>评估依据；</li>
 <li>评估内容和方法；</li>
 <li>结构检测结果；</li>
 <li>结构安全性、适用性和耐久性评估；</li>
 <li>评估结论，主要包括结构评估等级、存在的问题等；</li>
 <li>建议。</li>
 </ol>


 <p><a href="#idB.0.2.2"> B.0.2.2</a> <span id="idB.0.2.2">评估报告应由具备相关技术能力的专业技术人员编写；提交的报告应经审核人的审查核实和批准人的审查批准,并应有编写人、审核人和批准人的签字确认。</span></p>
 <p><a href="#idB.0.2.3"> B.0.2.3</a> <span id="idB.0.2.3">审核人和批准人应是具有相应资格能力的专业技术人员。</span></p>
 <p><a href="#idB.0.3"> B.0.3</a> <span id="idB.0.3">检测与评估报告应用词规范,各物理量和有效数值的计量单位应采用中华人民共和国法定计量单位,报告结论应明确。</span></p>
